## 数据库

### Postgres数据库

每个项目都带有一个完整的Postgres数据库。[文档](/docs/guides/database)

### 数据库扩展

每个数据库都有一套完整的Postgres扩展。[文档](/docs/guides/database/extensions)

### 数据库函数

创建可以从浏览器调用的自定义数据库函数。[文档](/docs/guides/database/functions)

### 数据库触发器

将触发器附加到数据表以处理数据库更改。[文档](/docs/guides/auth/managing-user-data#using-triggers)

### 数据库Webhook

使用Webhook将数据库更改发送到任何外部服务。<a href="https://supabase.com/blog/supabase-functions-updates#database-webhooks-alpha" target="_blank">链接</a>

### 数据库备份

每天备份项目，并可选择升级到时间点恢复。

### 搜索

使用Postgres全文搜索构建搜索功能。[文档](/docs/guides/database/full-text-search)

### 密钥与加密

用Postgres扩展Suabase Vault加密敏感数据并存储机密。<a href="https://supabase.com/blog/supabase-vault" target="_blank">链接</a>

### 数据库迁移

在本地进行开发，并使用数据库迁移将更改推送到生产数据库。[文档](/docs/guides/cli/local-development#database-migrations)

## 身份验证

### 电子邮件 & 密码登陆

为您的应用程序或网站构建电子邮件登录。[文档](/docs/guides/auth/auth-email)

### 魔术链接

为您的应用程序或网站构建无密码登录。[文档](/docs/guides/auth/auth-magic-link)

### 社交登陆

提供社交登录 - 从Apple到GitHub，再到Slack。[文档](/docs/guides/auth/social-login)

### 手机登陆

使用第三方短信提供商提供手机登录。[文档](/docs/guides/auth/phone-login)

### 行级安全

使用Postgres策略控制每个用户可以访问的数据。[文档](/docs/guides/auth/row-level-security)

### 服务器端身份验证助手

在流行的框架中实现用户身份验证的帮助程序，有如[Next.js](/docs/guides/auth/auth-helpers/nextjs)与[SvelteKit](/docs/guides/auth/auth-helpers/sveltekit)

### 身份验证UI工具包

使用自定义主题构建登录和注册页面。[文档](/docs/guides/auth/auth-helpers/auth-ui)

## API & 客户端库

### 自动生成REST API

RESTful API从您的数据库中自动生成，无需一行代码。[文档](/docs/guides/database/api#rest-api-overview)

### 自动生成GraphQL API

使用我们自定义的Postgres GraphQL扩展的快速GraphQL API。[文档](/docs/guides/database/api#graphql-api-overview)

### Realtime数据库变更

通过websockets接收数据库变更[文档](/docs/guides/realtime/postgres-changes)

### 用户广播

通过websockets在连接的用户之间发送消息。[文档](/docs/guides/realtime/broadcast)

### 用户存在感

在您的用户之间同步共享状态，包括在线状态和输入指示器。[文档](/docs/guides/realtime/presence)

### 客户端库

官方库[JavaScript](/docs/reference/javascript/)与[Dart](/docs/reference/dart)
非官方库[由社区支持](https://github.com/supabase-community#client-libraries)
