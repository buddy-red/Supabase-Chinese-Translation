此为Supabase为每个项目提供的功能非详尽列表

## 数据库

### Postgres数据库

每个项目都带有一个完整的Postgres数据库。[文档](/docs/guides/database)

### 数据库扩展

每个数据库都有一套完整的Postgres扩展。[文档](/docs/guides/database/extensions)

### Database Functions

创建可以从浏览器调用的自定义数据库函数。[文档](/docs/guides/database/functions)

### 数据库触发器

将触发器附加到数据表以处理数据库更改。[文档](/docs/guides/auth/managing-user-data#using-triggers)

### 数据库Webhook

使用Webhook将数据库更改发送到任何外部服务。<a href="https://supabase.com/blog/supabase-functions-updates#database-webhooks-alpha" target="_blank">链接</a>

### 数据库备份

每天备份项目，并可选择升级到时间点恢复。

### 搜索

使用Postgres全文搜索构建搜索功能。[文档](/docs/guides/database/full-text-search)

### 密钥与加密

用Postgres扩展Suabase Vault加密敏感数据并存储机密。<a href="https://supabase.com/blog/supabase-vault" target="_blank">链接</a>

### 数据库迁移

在本地进行开发，并使用数据库迁移将更改推送到生产数据库。[文档](/docs/guides/cli/local-development#database-migrations)

## 身份验证

### 电子邮件 & 密码登陆

Build email logins for your application or website. [文档](/docs/guides/auth/auth-email)

### 魔术链接

Build passwordless logins for your application or website.[文档](/docs/guides/auth/auth-magic-link)

### 社交登陆

Provide social logins - everything from Apple, to GitHub, to Slack. [文档](/docs/guides/auth/social-login)

### 手机登陆

Provide phone logins using a third-party SMS provider. [文档](/docs/guides/auth/phone-login)

### 行级安全

Control the data each user can access with Postgres Policies. [文档](/docs/guides/auth/row-level-security)

### 服务器端身份验证助手

Helpers for implementing user authentication in popular frameworks like [Next.js](/docs/guides/auth/auth-helpers/nextjs) and [SvelteKit](/docs/guides/auth/auth-helpers/sveltekit)

### 身份验证UI工具包

Build login and registration pages with custom themes. [文档](/docs/guides/auth/auth-helpers/auth-ui)

## API & 客户端库

### 自动生成REST API

RESTful APIs are autogenerated from your database, without a single line of code. [文档](/docs/guides/database/api#rest-api-overview)

### 自动生成GraphQL API

Fast GraphQL APIs using our custom Postgres GraphQL extension. [文档](/docs/guides/database/api#graphql-api-overview)

### Realtime数据库变更

Receive your database changes through websockets. [文档](/docs/guides/realtime/postgres-changes)

### User Broadcasting

Send messages between connected users through websockets. [文档](/docs/guides/realtime/broadcast)

### User Presence

Synchronize shared state across your users, including online status and typing indicators. [文档](/docs/guides/realtime/presence)

### 客户端库

Official client libraries for [JavaScript](/docs/reference/javascript/) and [Dart](/docs/reference/dart)
Unofficial libraries [supported by the community](https://github.com/supabase-community#client-libraries)
